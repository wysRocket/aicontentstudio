## Workflow

1. Select campaign context or objective.
2. Enter your prompt or use a saved template.
3. Generate multiple variants.
4. Edit tone, CTA, and channel-specific formatting.
5. Save or export your selected draft.

## Best practices

- Use specific audience and offer details in prompts
- Generate 3-5 variants before selecting one
- Keep a reusable prompt library for repeat campaigns
